# 受注経理担当者向け担当別手順

対象者: 請求担当者、入金担当者、経理担当者

最終更新日: 2026-06-17

## 1. 受注管理を開く

1. Agentify にログインします。
2. 「マイアプリ」を開きます。
3. 「受注管理」をクリックします。
4. 左メニューから「請求管理」「回収管理」「売掛台帳」を確認します。

![マイアプリから受注管理を開く](../assets/common/agentify-my-apps-order.png)

## 2. 表示される主な画面

| 画面 | 用途 |
| --- | --- |
| 受注一覧・受注詳細 | 請求対象の受注を確認します。 |
| 請求管理 | 請求一覧、請求詳細、請求作成を行います。 |
| 回収管理 | 入金登録、入金取消を行います。 |
| 売掛台帳 | 未回収残高、滞留状況を確認します。 |

受注の編集、受注確定、下書き削除は営業側の業務です。

![経理担当者の請求管理画面](../assets/roles/accounting-billing-view.png)

## 3. 請求を作成する

1. 受注一覧または受注詳細で請求対象を確認します。
2. 「請求作成」をクリックします。
3. 請求日、支払期限、請求金額を確認します。
4. 「請求作成」をクリックします。
5. 確認ポップアップで内容を確認し、登録します。

確認する項目:

- 宛先
- 請求日
- 支払期限
- 請求金額
- 消費税

![請求作成](../assets/order/billing-new.png)

## 4. 請求書を印刷・PDF保存する

1. 請求詳細を開きます。
2. 「請求書を印刷」をクリックします。
3. プレビューで内容を確認します。
4. 「印刷・PDF保存」をクリックします。
5. ブラウザの印刷画面で印刷または PDF 保存を行います。

請求取消済みのデータは、請求書として送付しないでください。

![請求書印刷ボタン](../assets/order/invoice-print-button.png)

## 5. 入金を登録する

1. 請求詳細または回収管理から「回収登録」をクリックします。
2. 入金日、入金金額、入金方法を入力します。
3. 必要に応じて振込名義、メモを入力します。
4. 「回収登録」をクリックします。
5. 確認ポップアップで内容を確認し、登録します。

入金後、請求の入金ステータスと売掛残高が更新されます。

![回収登録](../assets/order/collection-new.png)

## 6. 売掛台帳を確認する

1. 左メニューから「売掛台帳」を開きます。
2. 取引先、支払期限、売掛残高、滞留日数を確認します。
3. 必要に応じて「CSV出力」をクリックします。

主な入金ステータス:

| ステータス | 意味 |
| --- | --- |
| 未入金 | 入金がありません。 |
| 一部入金 | 請求額の一部が入金済みです。 |
| 入金済 | 全額入金済みです。 |
| 遅延 | 支払期限を過ぎています。 |
| 過入金 | 請求額を超えて入金されています。 |

![売掛台帳](../assets/order/ar-ledger.png)
