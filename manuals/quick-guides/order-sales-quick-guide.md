# 受注営業担当者向けクイック手順

対象者: 営業担当者、受注作成担当者

最終更新日: 2026-06-17

## 1. 受注管理を開く

1. Agentify にログインします。
2. 「マイアプリ」を開きます。
3. 「受注管理」をクリックします。
4. 左メニューから「受注管理」を開きます。

![マイアプリから受注管理を開く](../assets/common/agentify-my-apps-order.png)

## 2. 表示される主な画面

| 画面 | 用途 |
| --- | --- |
| 受注一覧 | 自分または自部門の受注を確認します。 |
| 受注詳細 | 受注内容、請求状況、入金状況を確認します。 |
| 新規受注作成 | 新しい受注を登録します。 |

請求管理、回収管理、売掛台帳が表示されない場合があります。これは請求・入金担当者向けの画面です。

![受注作成担当の画面](../assets/roles/order-creator-view.png)

## 3. 新規受注を作成する

1. 受注一覧で「新規受注」をクリックします。
2. 取引先、案件名、受注日、担当者、金額情報を入力します。
3. 明細に商品・サービス名、数量、単価、税率を入力します。
4. 金額と税額を確認します。
5. 「受注登録」をクリックします。
6. 確認ポップアップで内容を確認し、登録します。

下書き保存が表示されない場合、そのユーザーは正式登録のみ可能です。途中保存が必要な運用では、管理者に権限設定を確認してください。

![新規受注作成](../assets/order/order-new.png)

## 4. AI抽出を使う場合

1. 新規受注作成画面で「資料アップロード・抽出」をクリックします。
2. 注文書、見積書、契約書などのファイルを選択します。
3. 抽出結果がフォームに反映されたら、必ず内容を確認します。

確認する項目:

- 取引先名
- 案件名
- 受注日、納品予定日
- 明細の数量、単価、税率
- 合計金額

AI抽出は入力補助です。抽出結果を確認せずに登録しないでください。

![受注AI抽出ボタン](../assets/order/order-ai-extract-button.png)

## 5. 受注詳細を確認する

1. 受注一覧から対象の受注を開きます。
2. 受注番号、取引先、案件名、受注金額を確認します。
3. 請求状況、入金状況を確認します。

主なステータス:

| ステータス | 意味 |
| --- | --- |
| 下書き | 登録前の受注です。 |
| 受注済 | 正式登録済みで、請求前です。 |
| 請求済 | 請求データが作成されています。 |
| 回収済 | 入金まで完了しています。 |

![受注詳細](../assets/order/order-detail.png)

## 6. 困ったとき

| 状況 | 対応 |
| --- | --- |
| 受注管理がマイアプリに表示されない | Agentify 側のアプリ利用権限を管理者に確認します。 |
| 新規受注ボタンが表示されない | 受注作成権限を管理者に確認します。 |
| 請求作成ボタンが表示されない | 請求担当者の業務です。経理担当者へ依頼します。 |
| 登録後に内容を修正したい | 受注更新権限が必要です。営業責任者または管理者に確認します。 |
