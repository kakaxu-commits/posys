# クイック手順書 一覧

対象読者: 日常業務で受注管理・発注管理を利用する一般ユーザー、承認者、経理担当者

最終更新日: 2026-06-17

このフォルダの手順書は、操作説明を最小限にまとめた役割別の短い手順書です。詳しい項目説明、権限仕様、例外対応は各アプリの操作マニュアルを確認してください。

## 使い分け

- 初回研修では、このクイック手順書を使って担当業務だけを説明します。
- 詳細な画面項目や例外対応は、[受注管理 操作マニュアル](../order-management-manual.md) または [発注管理 操作マニュアル](../purchase-management-manual.md) を参照します。
- ロールや権限の考え方は、[権限・ロール説明書](../role-permission-guide.md) を参照します。
